Primeiros Passos

Para começarmos a codar em HTML e CSS iremos utilizar o aplicativo Visual Studio, que é um editor de codigo fonte,que nada mais é que uma ferramenta criada pela microsoft, e é utilizada para facilitar a criação de codigos.

Passo a passo

1º - Crie uma pasta vazia em seu dispositivo, para que se possa adicionar o conteúdo do codigo.
2º - Abra o Visual Studio Code e na aba explorador clique em "abrir pasta" e abra recém criada.
3º - Dentro da pasta recém criada clique em novo arquivo, e crie um nome para seu arquivo,que obrigatoriamente no final deve ter ".html" para indicar para sua plataforma de busca que o site é em HTML.
4º - Dentro da pasta recém criada adicione um "!" para criar o esqueleto do codigo em html
5º - Modifique a linguagem do site para "pt-br" para indicar ao navegador é em português
6º - Modifique o title do seu site por um que você desejar e pronto, você já esta apto para adicionarmos conteúdo ao site.
