Introdução

Existem três tipos de listas em HTML, as listas ordenadas, listas não ordenadas, e listas mistas.
Além disso as lista possuem atributos, que são comandos que utilizamos para complementar as listas, as deixando mais estilizadas.

Listas Ordenas <OL>

Listas Ordenadas são listas que obrigatoriamente devem seguir uma ordem. Por exemplo uma lista de carros mais rapidos do mundo.
Para se criar uma lista ordenada adicionamos o comando <ol> para indicar que estamos criando uma lista ordenada, e dentro dessa lista adicionamos o comando <li> para adicionar um item dentro dessa lista.
Podemos também adicionar o comando type dentro do comando ol para adicionar um atributo, e existem os seguintes atributos para uma lista ordenada:
-1-numerais
-A-letras maiusculas
-a-letras minusculas
-i-numeros romanos em letra minuscula
-I-numeros romanos em letras maiuscula
Além disso podemos indicar de que número será começada a contagem, adicionando o comando "start", e indicando o número que ira ser o ponta pé inicial da contagem, dentro do comando recém adicionado.

Listas Não Ordenadas <UL>

Listas não ordenadas são listas que não seguem um tipo específico de ordem. Por exemplo, uma lista de compras.
Para criar uma lista não ordenada, usamos o comando <ul>, e dentro dela adicionamos o comando <li> para adicionar os itens.
Para alterar o estilo dos marcadores, podemos usar o atributo "list-style-type" dentro do CSS. Os estilos disponíveis são:
- disc: adiciona um disco preenchido,
- circle: adiciona um círculo vazio,
- square: adiciona um quadrado.

Listas Mistas

São listas que misturam os dois tipos de listas mostrados anteriormente, as <UL> e as <OL>, colocando uma dentro da outra, não importando se é uma <UL> dentro de uma <OL> ou o contrario, podendo até mesmo adicionar os atributos na <UL> e <UL> normalmente.
Essas listas são muito utilizadas para a criação de menus
