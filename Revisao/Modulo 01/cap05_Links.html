Links Internos

São links que utilizamos para navegar entre as páginas de nosso proprio site, e para isso utilizamos a tag <A> e dentro del adicionamos a o nome da página para a qual desejamos navegar.
Por exemplo:
Para voltar para o capítulo inicial clique aqui.

Links Externos

São links que adicionamos de outro site em nosso site. E para isso adicionamos a tag <A> e em "href" adicionamos o link ao qual estamos acessando.
Por exemplo:
"Layla" - Erick Clapton

Links para download

São um metodo para criar um link para efetuar o download de um arquivo.
Para fazer isso temos que criar uma nova pasta para adicionarmos os arquivos que desejamos que os úsuarios possam realizar o download, e adicionarmos os arquivos. Após isso temos que criar uma tag <A> vazia (sem envelopar nenhuma palavra), em "href" fazemos o caminho para realizarmos o downloand, e adicionamos a tag "type" e adicionamos o texto necessario para realizar o download desse tipo de documento. Todos os textos necessarios para encontrarmos um item, encontramos no site iana.org
Por exemplo:

  • Primeiros passos html.pdf
  • Primeiros passos html.zip

Parâmetros

Grandes empresas utilizam atributos para melhorar a navegação em seu site, indicando algumas coisas para seu navegador.

  • rel
    • Esse atributo indica ao site que tipo de relacionamento aquele link tera com seu site.
    • "prev": Indica que aquele link é anterior a aquele que estamos na sequencia de página de nosso site.
    • "next": Indica que aquele link é posterior o que estamos na sequencia de páginas de nosso site.
    • "nofollow": Indica que aquele link não tem ligação com nosso site.
  • target
    • Esse parâmetros indica ao navegador aonde seu link será aberto.
    • "_blank": Indica que esse link será aberta em uma nova aba.
    • "_self": Indica que esse link será aberto na mesma aba que estamos.
