Seletores

Class (.)

Em CSS, o seletor class é usado para aplicar estilos a múltiplos elementos HTML que compartilham o mesmo atributo class. A sintaxe para usar uma classe é preceder o nome da classe com um ponto(.).
Por exemplo:
.minhaClasse { color: blue; }
Aplicará a cor azul ao texto de todos os elementos com a classe “minhaClasse”.

Id(#)

O seletor id é usado para aplicar estilos a um elemento HTML específico que possui um atributo id único. A sintaxe para usar um id é preceder o nome do id com um cerquilha (#). Por exemplo, #meuID { color: red; } aplicará a cor vermelha ao texto do elemento com o id “meuID”.

combinadores de CSS

são usados para definir a relação entre seletores e aplicar estilos a elementos específicos dentro de outros elementos.

  • Seletor descendente( ): é usado para selecionar todos os elementos que são descendentes de um elemento específico, ou seja que estão dentro dele independentemente da profundidade. A sintaxe é simplesmente um espaço entre dois seletores. Por exemplo, div p seleciona todos os elementos p que estão dentro da div.
  • Seletor Filho (>): O seletor filho é usado para selecionar elementos que são filhos imediatos de um elemento específico. Isso significa que ele só aplica estilos aos elementos que estão diretamente dentro do elemento pai, ignorando qualquer elemento aninhado mais profundamente.

Pseudo Classe(:)

Uma pseudo-classe em CSS é uma palavra-chave adicionada a um seletor que especifica um estado especial do elemento selecionado. Elas permitem que você aplique estilos a um elemento não apenas com base na sua posição na árvore do documento, mas também em relação a fatores externos, como a interação do usuário ou o estado do elemento.

exemplo:

  • :hover: Aplica um estilo quando o usuário passa o cursor sobre o elemento.
  • :visited: Aplica estilo a um link que foi visitado.

Pseudo elemento(::)

Um pseudo-elemento em CSS é uma palavra-chave adicionada a um seletor que permite estilizar partes específicas de um elemento. exemplo:

  • ::before: Insere conteúdo antes do conteúdo real de um elemento.
  • ::after: Insere conteúdo após o conteúdo real de um elemento.
