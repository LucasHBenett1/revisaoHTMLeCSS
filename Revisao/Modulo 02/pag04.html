Modelo de caixas

O modelo de caixa (box model) em CSS é um conceito fundamental que descreve como os elementos HTML são representados como caixas retangulares na página. Cada caixa é composta por várias partes que determinam seu tamanho e espaçamento. Vamos detalhar cada uma dessas partes:

  • Conteúdo (Content): É a área onde o texto e as imagens aparecem. O tamanho do conteúdo é definido pelas propriedades width e height.
  • Preenchimento (Padding): É o espaço entre o conteúdo e a borda da caixa. O preenchimento é transparente e pode ser ajustado usando as propriedades padding-top, padding-right, padding-bottom e padding-left.
  • Borda (Border): É a linha ao redor do preenchimento e do conteúdo. A borda pode ter várias propriedades, como border-width, border-style e border-color.
  • Margem (Margin): É o espaço fora da borda da caixa, separando a caixa de outros elementos. As margens também são transparentes e podem ser ajustadas usando as propriedades margin-top, margin-right, margin-bottom e margin-left.

Tipos de caixa

  • Caixas de Bloco As caixas de bloco são elementos que ocupam toda a largura disponível do seu contêiner e sempre começam em uma nova linha. Elas respeitam as propriedades de largura e altura, e as margens, preenchimentos e bordas empurram outros elementos para fora do caminho.
  • Caixas Inline As caixas inline são elementos que ocupam apenas o espaço necessário para o seu conteúdo e não começam em uma nova linha. Elas não respeitam as propriedades de largura e altura, mas as margens, preenchimentos e bordas laterais afetam o layout ao redor.

Grouping TAG'S

As grouping tags são fundamentais para criar uma estrutura HTML bem organizada e semântica. Elas ajudam a melhorar a legibilidade do código, facilitam a aplicação de estilos com CSS e a manipulação com JavaScript, além de melhorar a acessibilidade para tecnologias assistivas

  • <header>: O <header> é um elemento semântico que representa a seção de cabeçalho de uma página ou de uma seção específica. Ele geralmente contém informações introdutórias ou de navegação, como logotipos, títulos, menus de navegação e outros elementos de cabeçalho.
  • <nav> O <nav> é utilizado para agrupar links de navegação que direcionam para outras partes do site ou para páginas externas. Ele é especialmente útil para criar menus de navegação, barras laterais e rodapés com links importantes.
  • <main> O <main> é um elemento semântico que representa o conteúdo principal de um documento. Ele deve conter o conteúdo que é único para a página e que não se repete em outras páginas, como artigos, seções principais de texto, imagens e vídeos.
  • <section> O elemento <section> é usado para definir seções temáticas de conteúdo em um documento. Ele é ideal para agrupar conteúdo relacionado que faz parte de um tema maior.
  • <article> O elemento <article> representa um conteúdo independente e auto-contido que pode ser distribuído de forma independente ou reutilizável. Isso inclui artigos de notícias, posts de blog, comentários, widgets interativos, entre outros.
    • <aside> <aside> Usado para conteúdo complementar que está relacionado ao conteúdo principal, mas não é essencial. Ideal para barras laterais, notas de rodapé, e informações adicionais
